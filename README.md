# Proyecto Quiz - Next.js

## Cómo instalar
1. Asegurate de tener Node.js 18+ y npm.
2. Instala dependencias:

```powershell
cd "c:\Users\Thiago\Desktop\final de diseño web"
npm install
```

## Ejecutar en desarrollo
```powershell
npm run dev
# Abrir: http://localhost:3000
```

## Scripts importantes
- `npm run dev` — desarrollo (hot-reload)
- `npm run build` — build para producción
- `npm run start` — iniciar en producción (después de `build`)

## Estructura mínima
- `app/` — páginas
- `components/` — componentes UI
- `styles/` — estilos (Tailwind + reglas)
- `data/preguntas.json` — datos del quiz
